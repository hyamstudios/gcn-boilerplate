# HYAM-GCN-STARTER

## Local Setup

1. create `.env` according to `.env-example`
2. use `yarn`

## Netlify Setup

1. create environment variables in deploy settings according to `.env-example`
2. create `GOOGLE_ANALYTIC` env var if using google analytic

## Contribution
